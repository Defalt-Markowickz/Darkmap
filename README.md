## Install 

### Step 1: pip install python-nmap

### Step 2: git clone https://github.com/DefaltFUCK/Darkmap3.gi

### Step 3: cd Darkmap3/

### Step 4: chmod +x Darkmap.py

### Step 5: ./Darkmap.py


## Examples

### ./Darkmap.py -ip 192.168.1.40

### ./Darkmap.py --target 192.168.1.40
